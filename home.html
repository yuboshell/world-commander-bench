1.2 The paradigm

A human commands agents in plain language; a language model turns the command into coordinated action — in real time, on a normal GPU.

1.3 Current status
  • The benchmark: StarCraft II with the clock unpaused; the score is execution accuracy — built, expanded, switched tech, set composition as ordered — not win rate, which measures the game, not the executor.
  • The method — semantic parsing into a typed program (order→program): translate, don't evaluate — the model parses the order once; a deterministic interpreter does the state-dependent computation.
