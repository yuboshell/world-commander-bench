The paradigm

A human commands agents in plain language; a language model turns the command into coordinated action — in real time, on a normal GPU.

Leaf 1 · now
  • Free-form / abstract command — the commander's order in plain language, from concrete ("attack the left flank") to abstract ("hold the line, stay aggressive").
  • Interpreter (LLM) — turns that command into structured intent the agents can act on, fast enough for the live clock and within a memory budget.
  • Coordination (assignment / MARL) — splits the intent across the many agents (who does what), by assignment rules or a learned multi-agent policy.
  • Executor — game units: issues discrete commands to each unit (move, attack, use an ability) at the game's tick — the strategy-game domain (e.g. StarCraft II).
  • Executor — crowd: generates continuous full-body motion for each character (locomotion, gesture, holding formation), frame by frame — the crowd-motion domain.

Blue is the domain-agnostic shared middle (the budget wedge sits on the interpreter; coordination is the multi-agent layer). Red is the domain-specific executor. Green is where the tree is growing now — we add a box and arrow to the trunk for each leaf as it grows.
