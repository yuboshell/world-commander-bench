The paradigm

A human commands agents in plain language; a language model turns the command into coordinated action — in real time, on a normal GPU.

Current focus

Real-time crowd-motion command — grounding a command into a small crowd's coordinated full-body motion, in real time, on a budget.

Task 1 — close the real-time gap: swap E4's OmniControl (~16× too slow) for a real-time controllable model (MotionLCM / DART / CAMDM), and measure the frontier — FPS, latency, VRAM, grounding, FID — on yubopc.
